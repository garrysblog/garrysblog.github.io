Image to RGB565 Array Converter for Arduino

Overview

This converter transforms regular image files (PNG, JPG, GIF, etc.) into C/C++ arrays that Arduino can store in flash memory and display on RGB565 LCD screens.

What It Does

The converter performs the following operations:

  • Reads your image - Accepts any common format (PNG, JPG, GIF, BMP)
  • Extracts pixel data - Gets the color value of every pixel
  • Converts colors - Transforms from RGB888 (24-bit) to RGB565 (16-bit)
  • Generates C code - Creates a properly formatted C array with hex values
  • Outputs a header file - Produces a .h file ready to include in Arduino projects using the LovyanGFX library
